## 🌟 Projet Final : Application Web Django avec Analyseur de Texte

C'est le projet principal de ce dépôt. Il s'agit d'une application web complète développée avec le framework **Django**, intégrant un design professionnel et des fonctionnalités dynamiques.

### 📂 Emplacement du projet
Le projet se trouve dans le dossier : `TP_Final_Django/`

### 🛠️ Fonctionnalités clés
- **Authentification complète** : Inscription, Connexion et Déconnexion des utilisateurs avec gestion des messages d'erreur.
- **Analyseur de texte** : Un outil dynamique permettant de compter les mots d'un texte saisi par l'utilisateur.
- **Gestion de contenu (ORM)** : Affichage dynamique des services (Features) récupérés depuis la base de données SQLite.
- **Interface Professionnelle** : Intégration du template "OnePage" (Bootstrap 5) avec gestion des fichiers statiques.

### 📺 Démonstration Vidéo
Une vidéo montrant le fonctionnement complet de l'application est disponible ici :
👉 **[Voir la démonstration vidéo](./demo_projet.mp4)**

### 🚀 Installation et Lancement
1. Accéder au dossier : `cd TP_Final_Django`
2. Activer l'environnement virtuel : `.\venv\bin\activate`
3. Lancer le serveur : `python manage.py runserver`
4. Accéder à l'application sur : `http://127.0.0.1:8000/`
